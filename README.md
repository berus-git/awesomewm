# AwesomeWM ablakkezelő
- AwesomeWM konfigurációs állomány: rc.lua ( ~/.config/awesome/rc.lua ).
- Automatikusan induló programok: autostart.sh ( ~/.config/awesome/autostart.sh ).
- AwesomeWM rendszertálca szimbolikus ikonok színe: gtk.css ( ~/.config/gtk-3.0/gtk.css ).
- AwesomeWM Catppuccin téma: theme.lua_catppuccin

Előfordulhat, hogy a rendszertálcán az ikonok hibásan jelennek meg (pl. fekete alapon feketével), ekkor van szükség a gtk.css létrehozására.
